=== WORKING LOCALLY ===
First of all, to set up git from a local folder to the remote one on git I need to set up some kind of SSH authentication (Read git service FAQ)

To synchronize local folder with git folder:
  --> git pull --rebase origin main
  --> (optional, do if adding new file) git add ###
  --> git commit -m <message>
  --> git push -u origin main

To remove a file/a directory:
  --> Same as before, but insted of git add, do
  --> git rm (-r <directory>) <file>
